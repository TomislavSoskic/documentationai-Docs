## Overview

Docs start as private by default on Tomislav Soskic.

## Make Project Public

1. Go to your project dashboard.
2. Select the project you want to share.
3. Click "Settings".
4. Enable "Public visibility" toggle.
5. Copy your public link to share.

## Note

Only admins can change visibility settings.
