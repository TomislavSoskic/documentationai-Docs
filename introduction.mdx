## Overview

Tomislav Soskic provides a centralized platform where you organize and manage all your project documentation. Create structured spaces for codebases, APIs, user guides, and more. Access powerful features like search, versioning, and collaboration to keep your docs always up-to-date and accessible.

## Quick Start

Get up and running in minutes with these essential steps.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://tomislavsoskic.com` and create your first documentation space.
  </Step>
  <Step title="Add Your First Project" icon="folder-plus">
    Click "New Project" and name it, such as "My API Docs".
  </Step>
  <Step title="Upload or Create Docs" icon="upload">
    Drag and drop MDX files or use the editor to start writing.
  </Step>
  <Step title="Publish and Share" icon="globe">
    Hit publish to generate a live site. Share the URL with your team.
  </Step>
</Steps>

<Callout kind="tip">
  Start small: Create one project with a single `introduction.mdx` file to familiarize yourself with the editor.
</Callout>
