## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: Git installed for project imports

</Callout>

<Callout kind="tip">

Tomislav Soskic works best on desktop browsers. Mobile support is limited to viewing.

</Callout>

## Sign Up and Log In

Create your account in under a minute.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Details" icon="user">
    Provide your email, name, and a strong password. Accept the terms.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link from Tomislav Soskic and click it.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and access your workspace.
  </Step>
</Steps>

## Create Your Default Documentation Space

Your default space organizes all project docs.

<Steps>
  <Step title="Select New Space" icon="plus">
    From the dashboard, click **New Documentation Space**.
  </Step>
  <Step title="Name It" icon="edit-3">
    Enter "My First Project" as the name and choose a slug like `my-first-project`.
  </Step>
  <Step title="Customize Settings" icon="settings">
    Set visibility to public or private, and add a description.
  </Step>
  <Step title="Create Space" icon="check-circle">
    Click **Create**. Your space is ready at `https://docs.example.com/my-first-project`.
  </Step>
</Steps>

## Import Initial Project Files

Populate your space quickly using Git or API.

<Tabs>
  <Tab title="Git Import" icon="git-branch">
    <Steps>
      <Step title="Get Repo URL">
        Copy your GitHub repo URL, e.g., `https://github.com/yourusername/my-project`.
      </Step>
      <Step title="Connect Repo">
        In File Tree, click **Import > Git** and paste the URL.
      </Step>
    </Steps>
  </Tab>
  <Tab title="CLI Upload" icon="terminal">
````bash
npx tomislav-soskic upload --space=my-first-project --path=./docs --token=YOUR_API_TOKEN
````
    Replace `YOUR_API_TOKEN` with your token from dashboard settings.
  </Tab>
  <Tab title="API" icon="zap">
    <ParamField header="Authorization" param-type="string" required="true">
      Bearer `YOUR_API_TOKEN`
    </ParamField>
    <ParamField body="files[]" param-type="array" required="true">
      Array of file objects with path and content.
    </ParamField>

````javascript
const response = await fetch('https://api.example.com/v1/spaces/my-first-project/files', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    files: [{ path: 'quickstart.mdx', content: '# Hello World' }]
  })
});
````
  </Tab>
</Tabs>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Upload single file
const formData = new FormData();
formData.append('file', fileBlob, 'quickstart.mdx');
await fetch('https://api.example.com/v1/files', { method: 'POST', body: formData });
```

```python
import requests

files = {'file': ('quickstart.mdx', open('quickstart.mdx', 'rb'))}
response = requests.post('https://api.example.com/v1/files', files=files)
```
</CodeGroup>
